## Informations generales
- **Date de mise en ligne** : 02/12/2025
- **Environnement concerne** : Production

- **Technologie principale** : Symfony 7.x
- **Deploiement via** : GitHub (CI/CD)

---

## Objectifs de cette version

Premiere mise en ligne officielle de la plateforme afin de permettre aux utilisateurs de :
- Creer un compte utilisateur (nom, adresse email, mot de passe, photo de profil)
- Se connecter / se deconnecter
- Utiliser la fonction "Se souvenir de moi"
- Reinitialiser leur mot de passe via email
- Consulter les fiches d'animaux disponibles a l'adoption
- Ajouter des animaux en favoris
- Contacter les associations
- Filtrer les animaux par espece, sexe et departement
- Commenter les fiches animaux

---

## Contenu de la release

### Fonctionnalites livrees

**Authentification et gestion utilisateur**
- Systeme d'enregistrement d'un nouvel utilisateur avec verification de l'email
- Connexion utilisateur avec gestion de session
- Option "Se souvenir de moi" fonctionnelle via cookie securise
- Fonction de reinitialisation de mot de passe avec envoi d'email
- Upload de photo de profil avec VichUploader
- Page profil utilisateur avec gestion des favoris

**Gestion des animaux**
- Affichage des fiches animaux avec photos
- Filtres dynamiques (espece, sexe, departement)
- Carousel des 10 animaux les plus anciens sur la page d'accueil
- Page de detail d'un animal avec toutes ses informations
- Tags comportementaux (sociable, calme, joueur, etc.)
- Images par defaut selon l'espece si pas de photo

**Systeme de favoris**
- Ajout/retrait d'animaux en favoris via AJAX
- Boutons coeur avec animation
- API REST pour gestion des favoris (/api/favorites)
- Section "Mes Favoris" dans le profil utilisateur
- Notifications toast pour feedback utilisateur

**Systeme de commentaires**
- Ajout de commentaires sur les fiches animaux
- Commentaires imbriques (reponses aux commentaires)
- Affichage de l'auteur et de la date
- Formulaires de reponse dynamiques

**Gestion des associations**
- Page de presentation des associations
- Systeme d'adhesion a une association
- Validation des demandes d'adhesion par les gerants
- Roles hierarchises (Gerant, Membre, Utilisateur)
- Ajout d'animaux par les membres d'association
- Gestion des animaux de l'association

**Pages et navigation**
- Page d'accueil avec carousel et presentation
- Page liste des animaux avec pagination
- Page de detail d'un animal
- Page liste des associations
- Page profil utilisateur
- Menu responsive mobile
- Integration d'un design moderne et accessible

### Configuration technique
- Symfony 7.x
- Base de donnees MySQL
- Authentification geree via security.yaml (form_login + remember_me)
- VichUploader pour la gestion des fichiers
- Webpack Encore pour la compilation des assets
- JavaScript ES6 avec modules
- SCSS pour les styles
- API REST pour les interactions AJAX
- Variables d'environnement .env.local renseignees

## Procedure de mise en ligne

1. Retrait du /build dans le gitignore
2. Deploiement declenche via GitHub
3. Execution des commandes :
   ```bash
   composer install
   npm install
   cp .env .env.local
   php bin/console doctrine:database:create
   php bin/console doctrine:migrations:migrate
   npm run build
   php bin/console cache:clear --env=prod
   chmod -R 777 var/cache var/log public/uploads
   ```

## Base de donnees

### Tables principales
- user : Utilisateurs de la plateforme
- animals : Fiches animaux a l'adoption
- association : Associations partenaires
- association_member : Membres des associations
- species : Especes d'animaux
- tags : Tags comportementaux
- favorites : Table de liaison pour les favoris
- comments : Commentaires sur les animaux
- department : Departements francais
